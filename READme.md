## Project Description

This project is an interactive Machine Learning dashboard developed using **Python** and **Streamlit**.

It performs data preprocessing, trains a **Linear Regression** model, evaluates performance using metrics such as **R², RMSE, MSE, and MAE**, and visualizes insights through interactive charts and graphs.

The dashboard allows users to:
- Explore built-in datasets like **California Housing** and **Diabetes**
- Upload their own **CSV** files
- Adjust preprocessing settings (feature selection, scaling, train/test split)
- Analyze model performance effectively through interactive visualizations

---

## Features

- **Dataset Overview** — Sample rows, descriptive statistics, missing value report, and target distribution chart
- **Preprocessing** — Feature selection, missing value imputation, StandardScaler normalization, and correlation heatmap
- **Model & Evaluation** — Linear Regression training with R², RMSE, MSE, and MAE metrics
- **Visualizations** — Actual vs Predicted chart, Residuals analysis, Feature Importance, and Learning Curve
